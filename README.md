# The Bubbling Cauldron Full Stack Challenge

The brewery's potion order management system is broken. Orders are piling up, customers are furious, and 47 Healing Elixirs are stuck in Quality Control with no way to move them forward.

We need you to fix this.

**Use AI freely** — use whatever tools you would use to get work done in a real-world scenario.

## Getting Started

Choose one backend stack. All three are equivalent in scope and difficulty.

### Node.js Backend

**Requires:** Node 18+ and npm

```bash
npm run setup
npm start
```

### Python Backend

**Requires:** Node 18+, npm, and Python 3.10+

```bash
npm run setup:python
npm run start:python
```

### .NET Backend

**Requires:** Node 18+, npm, and .NET 10 SDK

```bash
npm run setup:dotnet
npm run start:dotnet
```

## The Challenge

Complete the following quests in order:

### Quest 1

The alchemist profile's service date validation is broken.

### Quest 2

Dragging potion orders between production stages doesn't work.

### Quest 3

Allow users to undo potion order changes, with a preview of what would be undone before confirming.

You decide the scope: how many levels of undo, and what actions can be undone.

**Before you start coding:** explain your approach. What would you implement, and why?
